# React Todo App

## Description

A simple todo app built with React.

## Tech Stack

- React
- Redux
- React Router
- Vite
- Tailwind CSS
- Yarn

## Features

- Add todo
- Delete todo
- Edit todo
- Toggle todo
- Filter todo
- Clear all todos
- Persist todos to localStorage
- Responsive design
- Set expiration date for todos as optional
- Set priority for todos as optional
- Show todos in a table
- Sort todos by expiration date
- Sort todos by priority
- Filter todos by priority
- Filter todos by expiration date
- Filter todos by text
